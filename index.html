Cisco RADIUS Cisco-AVPair 1 Multi-purpose attribute used for various functions including shell privilege levels, ACL assignments, VLAN assignments, SGT assignments, QoS policies, and bandwidth control.

Both Cisco-AVPair = "shell:priv-lvl=15" for admin access; Cisco-AVPair = "ip:inacl#10=permit ip 10.0.0.0 0.255.255.255 any" for ACL; Cisco-AVPair = "cts:security-group-tag=3" for SGT assignment
Implementation Steps
  1. Configure the RADIUS server to return this VSA with the appropriate formatting
  2. For ACLs: Use format "ip:inacl#="
  3. For admin level: Use format "shell:priv-lvl=" (levels 0-15)
  4. For SGT: Use format "cts:security-group-tag="
  5. For QoS: Use format "ip:sub-qos-policy-in="
  6. For VLAN: Use format "device-traffic-class="
  7. For VPN: Use format "vpn:addr-pool="
  8. Multiple attributes can be sent for different functions

  • Bandwidth Control: Adjust QoS parameters based on usage
